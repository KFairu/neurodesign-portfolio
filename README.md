# SH:OT вайб-дизайн

Локальный блог на Vite + React. Сейчас внутри один сайт с двумя опубликованными материалами:

- `/` — «Контролирование пространства и планировки. Часть 1»
- `/kontrolirovanie-prostranstva-i-planirovki-chast-2` — «Контролирование пространства и планировки. Часть 2»

Интерактивные схемы лежат в `public/drag-canvas*` и открываются внутри статей через iframe.

## Локальный запуск

```bash
npm install
npm run dev
```

Локальный адрес:

```text
http://127.0.0.1:5173/
```

## Проверка сборки

```bash
npm run build
npm run preview
```

После сборки Vite создает папку `dist`. Ее не нужно коммитить: Vercel соберет проект сам.
